# Chatbot Admin Frontend

Admin dashboard for managing chatbots built with React, Vite, and Tailwind CSS.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will run on `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- **Create Chatbot**: Build and configure new chatbots
- **My Chatbots**: View and manage all your chatbots
- **Appearance**: Customize chatbot appearance and settings
- **Preview**: Live preview of chatbots
- **Settings**: Manage account and application settings

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios
- Lucide React (Icons)
